Optimization techniques — executive cheat sheet

Use imperative verbs: "Summarize" not "Could you please summarize"
Use structured output: JSON/lists beat narrative explanations
Specify limits: "3 bullet points max" prevents output token bloat
Remove filler: delete "please", "certainly", "be sure to"
Use abbreviations: "pos/neg/neu" vs "positive/negative/neutral"
One good example beats 200 words of instruction — use few-shot

Token economics as strategic ROI decisions

Token spend is a capital allocation problem. The question is never "how do we reduce tokens?" — it's "where does token quality create the most business value?"
